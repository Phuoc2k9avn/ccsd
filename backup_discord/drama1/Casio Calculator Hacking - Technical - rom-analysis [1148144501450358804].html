  • 00: Triggered on high to low transition (falling edge)
  • 01: Triggered on low to high transition (rising edge)
  • 10: Continuously triggered when input is high level
  • 11: Continuously triggered when input is low level
Casio defaults F018H to 3, meaning it continuously triggers when a key is pressed. 3. SFR descriptions related to the keyboard:
  • F040H: KI, where high level input is 1 and low level input is 0
  • F041H: Sets whether the corresponding KI is connected to a pull-up resistor. 1 means disconnected, 0 means connected (default is 0)
  • F042H: Sets whether the corresponding KI can be used to trigger an interrupt. KIs set to 0 are masked.
  • F044H-F045H: Sets whether the corresponding KO is controlled by F046H-F047H. KOs set to 1 are masked.
  • F046H-F047H: KO, where a corresponding KO set to 1 is pulled down. The function of F047H.7 is unknown.
